# Todobackend

TODO app built using Elixir Plug and Cowboy


#### To Run

```sh
git clone https://github.com/saybhat/Elixir_cowboy_plug.git
cd  todo_submit
mix deps.get
mix ecto.create
mix ecto.migrate
iex -S mix
```
